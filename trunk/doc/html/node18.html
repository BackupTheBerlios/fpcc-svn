Bandwith settings for codecs

Within the file "codebook.xml", bandwith parameters of used codecs can be set.

The parameters have the following meaning:

  • encodingName: the name of the codec as used within the SDP payload
  • payloadType: default type value as used within SDP (optional)
  • type: can be audio or video
  • bandwith: needed bandwith, inclusive all overhead
