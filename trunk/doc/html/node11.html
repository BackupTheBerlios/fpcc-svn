Prerequisites

For a computer to run BIQINI, the following tools have to be installed:
  • ant
  • Java 1.6
  • traffic control
  • iptables
As the enforcing part uses TC and iptables, the PEP has to be run on a Linux machine. The PDP runs on any other operating system running Java 1.6.
