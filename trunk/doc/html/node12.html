Set-Up

Check out the source tree by means of subversion at

http://svn.berlios.de/svnroot/repos/fpcc/trunk

This gives us multiple folders:

  • bin: destination for compiled binaries
  • cfg: configuration files
  • doc: this documentation
  • lib: library folder for external libraries, e.g., log4j
  • src: source folder
